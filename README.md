# Playing with Java

An empty Java project that I use to try out things in Java.

## Useful Commands

- Set [Gradle Wrapper Version](https://docs.gradle.org/current/userguide/gradle_wrapper.html)

  ```shell
  $ ./gradlew wrapper --gradle-version=7.4.2
  ```

- Build project

  ```shell
  $ ./gradlew build
  ```

- Run tests

  ```shell
  $ ./gradlew check
  ```
